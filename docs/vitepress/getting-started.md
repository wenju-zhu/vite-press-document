[vitepress官网](https://vitepress.dev/guide/getting-started)

### 安装

```
npm install -g vitepress
```

### 创建项目

```
vitepress init my-vitepress-site
```

### 启动项目

```
cd my-vitepress-site
npm run docs:dev
```

### 打包项目

```
npm run docs:build
```
  
### 部署项目

```
npm run preview
```

### 配置

```
.vitepress/config.mts
```

### 目录结构

```
.
├── docs
│   ├── .vitepress
│   │   ├── config.mts
`
